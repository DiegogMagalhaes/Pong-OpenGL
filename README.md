#PONG OPENGL

Jogo Pong, criado usando c++ e OpenGL e GLUT.

#Flags para copilar projeto:
gcc -o pong main.cpp -lopengl32 -lglu32 -lfreeglut

#Como jogar:

Player 1:
a - move para cima
s - move para baixo

Player 2:
k - move para cima
l - move para baixo

Escolher dificuldade:
1- Easy
2- Normal
3- Hard
4- Very hard
5- Hyper hard
